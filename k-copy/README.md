# k-log 文件COPYDEMO
配置文件: /resources/config.txt，属性以=分隔

| 名称          | 说明                      |
| ------------- | ------------------------- |
| oldPath       | 复制之前文件夹路径        |
| newPath       | 复制之后文件夹路径        |
| newSuffix     | 新的后缀                  |
| newPrefix     | 新的前缀                  |
| newName       | 新的名称                  |
| needCopyNames | 需要复制的文件,以逗号分隔 |
| fileType      | 新的文件类型              |
